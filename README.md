# sampling-census
Sample analyses using data from the 2022 Brazilian Census

Este respositório organiza os estudos e produtos derivados do estudo em amostragem sob orientação da professora Alinne Veiga. A ideia é de usar dados do Censo de 2022 para verificar pontos da teoria de amostragem e apresentar resultados a partir disso. 

Projeto em desenvolvimento.
